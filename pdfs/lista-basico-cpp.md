<!-- new_lines: 1 -->

1. Percorra os números de 1 a 50
2. Para cada número, verifique se é par ou ímpar

# Exercício
<!-- new_lines: 1 -->

Escreva um programa que:
1. Aloque dinamicamente um array de inteiros com capacidade para 100 elementos
2. Crie uma função que receba o array (ponteiro) e uma variável de tamanho (por referência)
3. Na função, peça ao usuário para digitar o tamanho desejado e preencha o array com valores (ex: múltiplos de 10)
4. No main, exiba o array preenchido

# Exercício
<!-- new_lines: 1 -->

Escreva um programa que:
1. Crie uma template de função `trocar` que receba dois parâmetros por referência e troque seus valores
2. No `main`, crie dois `shared_ptr` para diferentes tipos (ex: `int` e `double`)
3. Use a função template para trocar os valores apontados pelos shared_ptr
4. Exiba os valores antes e depois da troca
5. Mostre a contagem de referências dos shared_ptr
